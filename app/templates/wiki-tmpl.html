Data Retrieval Endpoints (with example returns)

Note: All endpoints are prefixed by **/api/v1/**

Batch

Retrieve specified Batch by batch id or batch name

Endpoint: batches/{id}

Note: {id} represents either batch id or batch name. Business logic handles this for us.

Retrieve all members (trainees) for a given batch by specifying batch id

Endpoint: batches/{id}/members

Retrieve all Evaluations for a Batch at a given Week Endpoint: batches/{id}/week/{num}

Eval

Retrieve single Evaluation by ID

Endpoint: evaluations/{id}

Retrieve all Evaluations of a specified Trainee

Endpoint: evaluations/trainees/{id}

Retrieve all Evaluations of a specified Trainee at a specified Week

Endpoint: evaluations/trainees/{id}/week/{num}

Add new Eval

Endpoint: evaluations

Method: POST

Update Eval by ID

Endpoint: evaluations/update/{id}

Method: PUT

Delete Eval by ID

Endpoint: evaluations/delete/{id}

Method: DELETE

Person

Retrieve a single Person by Id

Endpoint: persons/{id}

Method: GET

Example:

persons/104

Retrieve page-able list of all Persons

Endpoint: persons/

Method: GET

Retrieve a page-able list of person/persons by first name and/or last name

Endpoints:

persons?firstname={firstname}

persons?lastname={lastname}

persons?firstname={firstname}&lastname={lastname}

Method: GET

Example:

persons?firstname=Efren

persons?lastname=Wakankar

Example:

persons?firstname=Tembong&lastname=Fonji

Retrieve persons by their role (trainer or trainee)

Endpoint: persons?role={role}

Method: GET

Example:

persons?role=2

Add a new person

Endpoint: persons

Method: POST

Return: The created Person

Update an existing person

Endpoint: persons

Method: PUT

Return: The updated Person

Delete an existing person

Endpoint: persons

Method: DELETE

Return: String message = {firstName} + " " + {lastName} + " was deleted."

Questions

Retrieve pageable list of all Questions

Endpoint: questions

Retrieve specific Question or list of Questions by Subject

Endpoints:

search?searchText={question text}

search?subject={subject name}

Method: GET

Add new Question

Endpoint: questions

Method: POST

Update Question by ID

Endpoint: questions/update/{id}

Method: PUT

Delete Question by ID

Endpoint: questions/delete/{id}

Method: DELETE
